Reflectie & Evaluatie

Terugkijken

Wat heb ik voor het aantonen van de leeruitkomsten gedaan?

Voordat ik aan mijn project begonnen ben, heb ik de leeruitkomsten goed doorgenomen en heb ik een idee voor een project bedacht dat deze leeruitkomsten zou moeten aantonen.
De eerste leeruitkomst was: "Trends en ontwikkelingen". Deze leeruitkomst heb ik aangetoond door een filmpje/vlog te maken van zoveel mogelijk technieken, producten en trends die in de stad te vinden zijn.
De volgende leeruitkomst is "Iteratief proces". Deze leeruitkomst heb ik aangetoond door mijn gekregen feedback te verwerken in mijn persoonlijke verantwoording. Je kunt daardoor goed zien wanneer ik vastliep, feedback heb gevraagd en oplossing heb bedacht!
Tot slot is de laatste leeruitkomst "Interactieve prototypes". Deze leeruitkomst heb ik aangetoond door mijn project interactief te maken. Ik heb geprobeerd mijn website zo interactief mogelijk te maken voor zover dit nu voor mij mogelijk is. Voor mijn project heb ik HTML, CSS en THREE JS gebruikt.

Wat heb ik geleerd?

Ik heb deze 7 weken erg veel geleerd. Voor deze periode had ik nog geen idee hoe ik een website in elkaar moest zetten, laat staan interactief maken! In deze 7 weken heb ik geleerd hoe je een website kunt maken met HTML en CSS. Ook heb ik geleerd hoe je deze interactief kunt maken. Ook heb ik me verdiept in de taal THREE JS. Hierin heb ik geleerd hoe je met 3D objecten kunt werken.
Naast mijn project heb ik ook geleerd wat media precies inhoudt. Ik heb ook gezien hoe vaak design en technologie samen gaan; als je naar de stad kijkt zie je deze combinatie véél vaker dan verwacht!

Waarin wil ik me nog verder ontwikkelen?

Ik zou me nog verder willen ontwikkelen in het maken van websites. Als ik een nieuwe website zou moeten maken, zou ik graag willen dat deze website responsief is. Ook zou ik me graag verder willen ontwikkelen in het maken van kleine animaties, transities en effecten op een website. Tot slot zou ik willen leren hoe je een website nog interactiever kunt maken!

Waar ben ik trots op?

Ik ben heel erg trots op het leerproces dat ik deze 7 weken heb gehad. Ik heb erg veel kunnen leren. Natuurlijk ben ik ontzettend trots op mijn gemaakte project. Ik had van te voren nooit gedacht dat mijn project er aan het einde van deze weken zo uit zou komen te zien!

Wat doe ik een volgende keer anders?

Ik merk dat ik soms erg veeleisend van mezelf voor mezelf ben. Ik denk dat ik dit de volgende keer iets minder moet zijn. Doordat ik het graag beter wil doen dan wordt gevraagd, kom ik soms problemen tegen die ik nog niet op kan lossen. Hier blijf ik dan (te) lang naar een oplossing zoeken, terwijl ik mijn tijd beter aan andere dingen had kunnen besteden.

Welke formatieve indicatie zou ik mezelf geven voor deze leeruitkomst?

Ik zou mezelf een 'Good' geven. Ik heb alle leeruitkomsten aangetoond en heb erg veel geleerd over het ontwikkelen van websites. Ik heb altijd serieus en enthousiast gewerkt waardoor ik altijd hard door kon blijven werken. Door deze manier van werken heb ik ook erg veel kunnen leren en maken. Ik ben zelf erg tevreden over wat ik in deze weken bereikt heb en op mijn eindproduct!
